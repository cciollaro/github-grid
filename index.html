Github Swagifyer




Use this tool to generate a custom display image for a Github profile page.

Click or drag the mouse in the above image to draw a "commit history" that will be exported as a series of commits varying throughout the past year timestamps.

If, for some bizarre reason, you aren't able to create an appealing image using just your mouse, you can also try your luck by using the "Upload Image" button above.

For best results when using a custom image, try to match the grid dimensions exactly.
